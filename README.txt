CARA PASANG DI ACODE ANDROID:
------------------------------

1. Download file ZIP: PERFOTOFOLIO.zip
2. Ekstrak ke: SD/CARD/DOWNLOAD/PERFOTOFOLIO/
3. Buka Acode → Open Folder → arahkan ke PERFOTOFOLIO/
4. Selesai

STRUKTUR FOLDER:
----------------
PERFOTOFOLIO/
├── index.html          (Beranda - Dark Theme)
├── about.html          (Tentang - Light Theme)
├── gallery.html        (Galeri - Light Theme)
├── contact.html        (Kontak - Light Theme)
├── css/
│   └── style.css       (Semua style + animasi)
├── js/
│   └── main.js         (Semua logic + animasi)
└── assets/
    └── images/         (Kosong - kamu isi foto)

KEMASUKAN GITHUB:
-----------------
1. Buka terminal di Acode
2. Ketik:
   git init
   git add .
   git commit -m "Initial portfolio"
   git branch -M main
   git remote add origin https://github.com/USERNAME/alif-febrian-portfolio.git
   git push -u origin main

3. Aktifkan GitHub Pages di Settings → Pages → main branch
